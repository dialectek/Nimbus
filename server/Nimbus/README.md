Nimbus server
======================
Build:
```
build.sh
```

Run server:

``` 
dilemma_server.sh
```

Run client:

```
dilemma_client.sh [<id>] [<server network address> (defaults to localhost)]
```

Client commands:

```
peer_message:<agent name>;<message>
agent_action:<agent_name>;cooperate|betray
agent_score
agent_clear
agent_scores
game_quit
```
