Nimbus server.

See Nimbus.java for the java API.
See rest/Nimbus.java for the REST API.

Server:
Import and build Eclipse project.
Export Nimbus.war tomcat servlet.
Deploy Nimbus.war to tomcat.

HTML:
Run Nimbus/src/main/webapp/nimbus.html in browser.

REST prefix for localhost: http://localhost:8080/Nimbus/rest/service/
